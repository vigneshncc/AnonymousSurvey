# Blockchain Survey
Blockchain survey allows user to take completely anonymous survey.

# Technologies used
Blockchain
Ethereum Smart Contracts
React

# How to configure

Pre-requisites.
RUN CMD as Administrator and run 
npm install --global windows-build-tools

Clone this repository,

Edit truffle-config.js and add, seed, rinkeby network ID in the placeholder given.

Do a npm install

CD to Client Directory.

Do a npm install there.

CD one step back.

Run the following command truffle deploy --network rinkeby --reset

Cd to Client Directory

npm run start.

Configure metamask chrome plugin and now you are ready to take survey.
